# Nova UI

Frontend implementation of the provided Figma design using React + Vite.

## Tech Stack

- React 19
- Vite
- CSS3

## Setup Instructions

1. Clone the repository

```bash
git clone https://github.com/Nandani-thakur/Nova-ui.git
Navigate to the project folder
cd nova-ui
Install dependencies
npm install
Start development server
npm run dev
Build for production
npm run build
